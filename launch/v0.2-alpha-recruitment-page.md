# 入行365 StyleOS Creator Alpha

StyleOS Creator Alpha 招募 20 位造型 / 穿搭 / 形象 / 发型 / 美妆创作者，一起测试 AI personal styling workflow 的开放协议和创作者工具。

## 招募对象

- 造型博主
- 穿搭博主
- 发型博主
- 形象顾问
- 美妆博主
- 摄影 / 妆造机构
- 小型形象服务团队

## 你会得到什么

- StyleOS v0.2 Hairstyle Module 文档包
- 发型信息采集表
- 发型轻量报告模板
- 理发师沟通卡
- 服务菜单模板
- 反馈表和验证清单
- 与入行365 StyleOS 团队共同打磨 Creator Studio / Cloud 方向的机会

## 你需要参与什么

- 试用 intake form。
- 试写或评审 lite report。
- 测试 barber brief 是否可执行。
- 反馈服务菜单是否清晰。
- 提供 1-3 个 synthetic 或 fully anonymized workflow notes。
- 参与一次反馈访谈。

## 适合谁

- 已经收到粉丝咨询的人。
- 想把经验产品化的人。
- 想做轻量付费服务的人。
- 愿意给早期工具提反馈的人。
- 认可隐私和授权边界的人。

## 不适合谁

- 只想要即时 C 端测试的人。
- 不愿意参与反馈的人。
- 希望公开提交真实粉丝照片的人。
- 希望把 starter rules 包装成专家认证的人。
- 希望直接获得 StyleOS Pro 商业库的人。

## 时间周期

- Week 1: onboarding and intake review
- Week 2: report testing
- Week 3: barber brief and service menu review
- Week 4: feedback synthesis

## 数据和隐私原则

- 不向 public repo 提交真实照片。
- 不提交真实姓名、联系方式、账号、聊天记录或付款记录。
- 不提交未经授权案例。
- 公开案例使用 synthetic examples。
- 私下反馈也应最小化、脱敏并获得授权。
