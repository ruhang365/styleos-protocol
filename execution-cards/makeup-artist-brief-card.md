# Makeup Artist Brief Card

This card helps users communicate makeup direction to a makeup artist.

中文说明：
化妆师沟通卡用于说明皮肤质感、眉形方向、眼妆方向、腮红和唇色方向，以及避雷点。

## Template

```yaml
execution_card_type: "makeup_artist_brief"
skin_finish: ""
eyebrow_direction: ""
eye_makeup_direction: ""
cheek_lip_direction: ""
avoid:
  - ""
scenario: ""
lighting_context: ""
```

## Field Notes

- Skin finish: clean natural, soft matte, soft glow, camera-ready.
- Eyebrow direction: clean order, soft arch, defined but not heavy.
- Eye makeup direction: low emphasis, soft definition, camera definition.
- Cheek/lip direction: color direction, placement, intensity.
- Avoid: over-heavy base, overly sharp contrast, product names if not needed.
- Scenario: daily, professional, camera, event.
