# Hairstyle Pro Boundary

This document defines the open-core boundary for the Hairstyle Module.

## Open Source Layer

The open repository includes:

- taxonomy;
- starter rules;
- synthetic cases;
- barber brief template;
- creator workflow;
- validation checklist;
- schema and report structures.

## Future StyleOS Pro Layer

The Pro layer may include:

- expert-reviewed hairstyle rules;
- verified case database;
- advanced face / hair mapping engine;
- creator-specific model;
- high-quality before / after feedback loop;
- salon partner execution network.

## Boundary Rule

Open content can explain the protocol and provide starter examples.

Pro content may include higher-confidence expert patterns, authorized data feedback, and commercial execution networks. It is not part of the Apache-2.0 repository license unless separately released.
