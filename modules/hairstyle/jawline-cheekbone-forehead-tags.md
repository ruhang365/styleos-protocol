# Jawline / Cheekbone / Forehead Tags

These local tags help creators choose more precise rule cards.

## `soft_jawline`

- 观察信号：下颌边界较柔和。
- 可能对应发型策略：可增加适度线条感来提升清晰度。
- 注意事项：不要把柔和描述为缺点。

## `strong_jawline`

- 观察信号：下颌线条或下颌角存在感较强。
- 可能对应发型策略：避开发尾正卡最宽点，或用柔和层次过渡。
- 注意事项：强线条也可以服务专业感。

## `wide_jawline`

- 观察信号：下半脸横向视觉更明显。
- 可能对应发型策略：平衡头顶和脸侧，不只靠遮挡。
- 注意事项：不要使用羞辱性语言。

## `narrow_chin`

- 观察信号：下巴视觉更集中或收窄。
- 可能对应发型策略：避免上半脸过重，适度平衡下半脸。
- 注意事项：不能单独决定发型。

## `visible_cheekbone`

- 观察信号：颧部线条或中面部宽度较明显。
- 可能对应发型策略：脸侧层次避开最宽点。
- 注意事项：颧骨存在感不是负面评价。

## `soft_cheek_area`

- 观察信号：脸颊区域曲线较柔和。
- 可能对应发型策略：增加轻线条或控制侧发量。
- 注意事项：避免“遮脸”作为默认策略。

## `high_forehead_impression`

- 观察信号：额头露出或上半脸高度明显。
- 可能对应发型策略：轻刘海、侧分、碎发平衡。
- 注意事项：不应默认厚重遮挡。

## `low_forehead_impression`

- 观察信号：额头留白较少。
- 可能对应发型策略：控制刘海厚度，提高清爽度。
- 注意事项：不代表不能留刘海。

## `balanced_forehead`

- 观察信号：额头比例相对稳定。
- 可能对应发型策略：可根据目标选择露额或轻遮挡。
- 注意事项：仍需考虑发际线和场景。

## `mixed_uncertain`

- 观察信号：角度、光线或输入不足导致判断不稳定。
- 可能对应发型策略：先给低风险方向。
- 注意事项：不确定就是有效判断，不要强行分类。
