# Hairstyle Mapping Matrix

This starter matrix maps signals to possible strategies. It is not deterministic and should be reviewed by creators.

中文说明：
这是 starter matrix，用于帮助创作者从输入信号推导可解释策略。它不是绝对判断，也不是专家认证库。

| Face / attribute signal | Common risk | Possible strategy | Avoid | Execution card note |
| --- | --- | --- | --- | --- |
| square jawline + goal softer | 下颌线条被发尾进一步强调 | 增加侧层次，让发尾避开下颌最宽处 | blunt chin-length cut | 在理发师沟通卡写明“发尾不要正好卡在下颌最宽处” |
| round-soft face + goal cleaner | 两侧发量贴脸或堆积，清爽度下降 | 增加纵向线条和轻量 face framing | 厚重齐刘海 + 两侧贴脸 | 说明希望“脸侧轻一点，头顶适度支撑” |
| long face + goal softer | 头顶过高让纵向更明显 | 降低 crown height，增加侧向柔和层次 | 过度垫高头顶 | 写明“不要继续拉高头顶” |
| high forehead + goal balance forehead | 露额过多或厚遮挡都可能不自然 | 用轻刘海、侧分或碎发平衡 | 厚重全遮挡 | 写明“轻遮挡，不要厚重盖住额头” |
| low hair volume + goal increase volume | 过度打薄后更贴头皮 | 保留支撑层，调整分线和头顶蓬松 | 过度打薄、贴头皮直发 | 写明“优先保留支撑，不要大量打薄” |
| high hair volume + goal cleaner | 体积过厚导致笨重 | 做重量管理和轮廓收束 | 一味剪短但不处理重量 | 写明“减轻重量感，而不是只剪短” |
| sharp features + goal softer | 强直线让距离感更明显 | 柔和层次、低对比线条 | 过硬切线和过强锐角 | 写明“线条柔和，不要太硬” |
| soft features + goal sharper | 全部圆线可能显得不够精神 | 增加适度线条和清楚分区 | 过度圆润或过强锐角 | 写明“增加秩序，但不要攻击感过强” |
| workplace scenario + low maintenance | 复杂造型难持续 | 清爽、稳定、易打理优先 | 高维护卷度和复杂层次 | 写明“每天 5-10 分钟可维护” |
| goal more mature + heavy bangs | 学生感可能偏强 | 刘海轻量化或侧分，提高露额秩序 | 厚重齐刘海 | 写明“减少学生感，保留清爽度” |
| goal younger + professional scene | 年轻感过度可能削弱可信度 | 轻盈但保持结构 | 只用齐刘海表达年轻 | 写明“年轻感不要牺牲职场稳定” |
| barber reference only + uncertain input | 理发师只看网图可能误解目标 | 描述目标、避雷、维护和长度边界 | 只给单张网图不说明原因 | 写明“按目标和限制沟通，不只给图片” |
