# Hairstyle Goal Tags

Hairstyle goal tags describe what the user wants the hairstyle to help with.

中文说明：
目标标签用于表达“想通过发型改善什么感受或场景表现”。目标之间可能冲突，真实服务中需要排序。

## `goal_look_cleaner`

- 目标含义：整体更清爽、利落、容易被理解。
- 适合的输出类型：减少发尾杂乱、控制两侧堆积、优化露额和分线。
- 可能冲突的目标：过度追求蓬松或复杂层次可能降低清爽度。
- 示例表达：想看起来更清爽，适合通勤和日常见人。

## `goal_look_softer`

- 目标含义：降低线条攻击性，增加柔和和亲和感。
- 适合的输出类型：柔和层次、侧分、自然弧度、降低强直线边界。
- 可能冲突的目标：想显非常锐利或强气场时需要取舍。
- 示例表达：希望少一点距离感，整体更温和。

## `goal_look_more_professional`

- 目标含义：更适合职场、会议或正式沟通。
- 适合的输出类型：清楚轮廓、稳定长度、低复杂度、可维护造型。
- 可能冲突的目标：过度学生感、过强潮流感或高维护风格。
- 示例表达：希望适合面试、客户沟通或正式工作场景。

## `goal_look_younger`

- 目标含义：降低沉重、疲惫或过度成熟的视觉感。
- 适合的输出类型：轻盈发尾、适度层次、柔和动态。
- 可能冲突的目标：显年轻不等于必须齐刘海，过度幼态可能影响专业感。
- 示例表达：想更轻松、有活力，但不想显得过度学生感。

## `goal_look_more_mature`

- 目标含义：更稳定、可信、正式。
- 适合的输出类型：减少碎乱、提高轮廓秩序、控制刘海学生感。
- 可能冲突的目标：过度柔软或过度可爱方向。
- 示例表达：希望看起来更稳、更适合负责型角色。

## `goal_look_sharper`

- 目标含义：增加线条感、精神感和辨识度。
- 适合的输出类型：更清晰的分线、利落发尾、适度减少圆润堆积。
- 可能冲突的目标：想显柔和或低攻击感时需要降低锐度。
- 示例表达：希望更精神、更有轮廓，但不要太强势。

## `goal_look_more_elegant`

- 目标含义：更安静、流畅、有秩序。
- 适合的输出类型：顺滑层次、低噪声轮廓、克制体积。
- 可能冲突的目标：强烈潮流感或过度凌乱感。
- 示例表达：想要更耐看、更有整理感。

## `goal_reduce_face_width_impression`

- 目标含义：降低两侧横向扩张感。
- 适合的输出类型：控制脸侧厚度、增加纵向线条、避免发尾堆在最宽处。
- 可能冲突的目标：想增加甜美圆润感时需要谨慎。
- 示例表达：希望脸侧不要显得太拥挤，整体更舒展。

## `goal_balance_forehead`

- 目标含义：调整额头露出比例或上半脸存在感。
- 适合的输出类型：轻刘海、侧分、碎发、发际线边缘修饰。
- 可能冲突的目标：极高露额清爽感或强正式感。
- 示例表达：希望额头比例更自然，但不想厚重遮挡。

## `goal_balance_jawline`

- 目标含义：让下颌线条与发型轮廓更协调。
- 适合的输出类型：避开发尾正卡下颌最宽处、用侧层次转移边界。
- 可能冲突的目标：想强调强结构感时未必需要柔化。
- 示例表达：希望下半脸线条和发尾过渡更自然。

## `goal_increase_volume_impression`

- 目标含义：增加头发支撑和蓬松感。
- 适合的输出类型：头顶支撑、轻层次、分线调整、避免过度贴头皮。
- 可能冲突的目标：低维护或想减少体积时需要取舍。
- 示例表达：希望头发看起来更有支撑，不要贴头皮。
