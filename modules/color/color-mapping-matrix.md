# Color Mapping Matrix

This starter matrix maps color signals to possible strategies.

中文说明：
色彩 mapping matrix 用来支持方向判断，不直接替代创作者人工判断。

| Signal | Common risk | Possible strategy | Avoid | Execution note |
| --- | --- | --- | --- | --- |
| low contrast face | 极强黑白撞色可能压过面部 | 使用柔和对比或中等对比组合 | 默认黑白强撞色 | 先输出“低到中对比方向” |
| warm tendency | 荧光冷色可能显突兀 | 选择暖中性或低刺激冷色点缀 | 大面积荧光冷色 | 标注为谨慎尝试 |
| cool tendency | 过黄过浊色可能显疲惫 | 选择冷中性或干净低饱和色 | 大面积黄浊色 | 先测试近脸色 |
| work scenario | 流行色波动大，稳定度不足 | 中性色作为主色，趋势色做小面积 | 全身强流行色 | 先确定主色和点缀色 |
| camera scenario | 下装颜色对面部影响较弱 | 优先管理上衣和近脸色 | 只优化下装 | 执行卡写“上半身优先” |
| high saturation preference | 容易抢过面部或场景 | 降低面积或放到配饰 | 大面积高饱和近脸 | 使用 10-20% 点缀 |
| muted preference | 可能显得没精神 | 增加一个清晰但不刺眼的焦点 | 全身灰浊无层次 | 加入中明度提气色 |
| uncertain undertone | 强行分类导致误导 | 输出中性安全方向并建议线下测试 | 直接绑定具体色号 | 标注为 uncertain |
