# Color Taxonomy

Color taxonomy organizes visual color dimensions used by StyleOS Protocol.

中文说明：
色彩 taxonomy 不用于给用户贴固定季型身份标签，而是帮助创作者输出更稳定的色彩方向。

## Dimension Groups

### Undertone Tendency

Possible tags:

- `undertone_warm_tendency`
- `undertone_cool_tendency`
- `undertone_neutral_tendency`
- `undertone_uncertain`

### Brightness

Possible tags:

- `brightness_low`
- `brightness_medium`
- `brightness_high`

### Saturation

Possible tags:

- `saturation_muted`
- `saturation_medium`
- `saturation_clear`
- `saturation_neon_risk`

### Contrast

Possible tags:

- `contrast_low`
- `contrast_medium`
- `contrast_high`

### Temperature Impression

Possible tags:

- `temperature_warm_impression`
- `temperature_cool_impression`
- `temperature_balanced_impression`

### Color Risk

Possible tags:

- `color_risk_overpowering`
- `color_risk_dullness`
- `color_risk_fatigue`
- `color_risk_camera_cast`

### Scenario Color

Possible tags:

- `scenario_color_work_stable`
- `scenario_color_camera_friendly`
- `scenario_color_social_accent`
- `scenario_color_event_polished`

## Pro Library Boundary

Detailed expert color systems, advanced model matching, and real-case verified palettes are outside the open seed pack unless separately released.
