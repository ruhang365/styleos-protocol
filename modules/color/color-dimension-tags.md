# Color Dimension Tags

Color dimension tags provide starter labels for compatible tools.

中文说明：
这些标签用于表达色彩方向，不等于固定商品色号，也不应强行绑定某个品牌或产品。

## Undertone Tendency

- `undertone_warm_tendency`: 整体更适合偏暖或暖中性色方向。
- `undertone_cool_tendency`: 整体更适合偏冷或冷中性色方向。
- `undertone_neutral_tendency`: 冷暖边界不强，适合中性策略。
- `undertone_uncertain`: 输入不足或判断不稳定。

## Brightness

- `brightness_low`: 低明度方向，适合更稳重或收敛的视觉。
- `brightness_medium`: 中明度方向，日常适配度高。
- `brightness_high`: 高明度方向，可能更轻盈但需要控制场景。

## Saturation

- `saturation_low`: 低饱和、柔和，适合克制或低刺激方向。
- `saturation_medium`: 中等饱和，稳定可用。
- `saturation_high`: 高饱和，存在感更强，需要结合场景和面积。
- `saturation_clean`: 清晰干净的饱和印象，适合需要精神感或镜头识别的场景。
- `saturation_muted`: 灰度更高、柔和收敛，适合日常、专业或低对比方向。
- `saturation_neon_risk`: 荧光或高刺激色，需要谨慎。

## Contrast

- `contrast_low`: 面部或整体适合低对比组合。
- `contrast_medium`: 可接受中等对比。
- `contrast_high`: 可承载更强黑白、深浅或冷暖对比。

## Scenario Color

- `scenario_color_work_stable`: 职场稳定色彩。
- `scenario_color_camera_friendly`: 镜头友好，减少偏色风险。
- `scenario_color_social_accent`: 社交或内容场景可增加记忆点。
- `scenario_color_event_polished`: 活动或正式场景更完整。

## Impression Tags

- `color_impression_clean`: 干净清晰，适合想提升精神感的场景。
- `color_impression_muted`: 柔和克制，适合低刺激、日常或专业场景。
- `color_camera_friendly`: 镜头友好，优先考虑上半身和近脸色。
- `color_professional`: 职场友好，优先稳定、可信和不过度分散注意力。
