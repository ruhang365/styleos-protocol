# Scenario Taxonomy

Scenario taxonomy defines starter context tags for styling workflows.

## Scenario Tags

### `scenario_daily_commute`

日常通勤。重点是低维护、舒适、稳定和可重复。

### `scenario_client_meeting`

客户会议。重点是可信、清晰、稳定，未必追求强风格。

### `scenario_interview`

面试。重点是专业、可信、不过度分散注意力。

### `scenario_date`

约会。重点是舒适、亲和、真实表达，避免过度表演化。

### `scenario_content_creation`

内容创作。重点是可记忆点、镜头识别和个人品牌一致性。

### `scenario_live_streaming`

直播。重点是上半身、灯光、背景、稳定识别。

### `scenario_photoshoot`

拍摄。重点是灯光、背景、镜头、上半身优先级和团队沟通。

### `scenario_wedding_guest`

婚礼宾客。重点是正式、得体、不过度抢主角。

### `scenario_public_speaking`

公开表达。重点是可信、稳定、远距离识别和动作舒适度。
