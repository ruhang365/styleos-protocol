# Accessories Module

The Accessories module structures glasses, hats, earrings, necklaces, bags, and accessory visual weight.

中文说明：
配饰模块用于把眼镜、帽子、耳饰、项链、包和视觉重量拆成可解释建议，避免只凭单品流行度推荐。
