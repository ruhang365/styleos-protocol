# Accessories Taxonomy

Accessories taxonomy defines starter dimensions for accessories recommendations.

## Glasses Shape

- `glasses_round`
- `glasses_square`
- `glasses_soft_rectangle`
- `glasses_cat_eye`
- `glasses_light_frame`
- `glasses_heavy_frame`

## Hat Brim

- `hat_brim_short`
- `hat_brim_medium`
- `hat_brim_wide`
- `hat_crown_low`
- `hat_crown_high`

## Earrings Scale

- `earrings_small`
- `earrings_medium`
- `earrings_large`
- `earrings_linear`
- `earrings_round`

## Necklace Length

- `necklace_short`
- `necklace_collarbone`
- `necklace_mid`
- `necklace_long`

## Bag Structure

- `bag_soft`
- `bag_semi_structured`
- `bag_structured`
- `bag_minimal`
- `bag_statement`

## Accessory Visual Weight

- `accessory_weight_light`
- `accessory_weight_medium`
- `accessory_weight_heavy`
