# Outfit Module

The Outfit module structures clothing direction for silhouette, fabric, neckline, shoulder line, waistline, length, formality, scenario, and personal brand goals.

中文说明：
穿搭模块帮助创作者把“适合怎么穿”拆成可解释、可执行、可复盘的方向，而不是直接给单一商品答案。
