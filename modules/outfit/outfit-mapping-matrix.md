# Outfit Mapping Matrix

This starter matrix maps outfit signals to possible strategies.

| Signal | Common risk | Possible strategy | Avoid | Execution note |
| --- | --- | --- | --- | --- |
| client meeting + trustworthy goal | 过强潮流感降低稳定度 | 中性色 + 轻结构版型 | 只用全黑西装表达专业 | 区分可信和压迫感 |
| soft-rounded style + professional goal | 过软面料显得不够利落 | 提高面料挺括度和肩线清晰度 | 全身松软无结构 | 先调整外套和上衣 |
| sharp style + approachable goal | 线条和颜色都过强 | 降低颜色对比，保留少量结构 | 强肩线 + 高对比全套 | 保留专业，降低攻击性 |
| low maintenance user | 多层复杂搭配难复现 | 2-3 件核心公式 | 过度叠穿 | 输出可重复公式 |
| camera scene | 下半身优化不明显 | 优先上半身轮廓、领口和近脸色 | 忽视领口 | 执行卡写“上半身优先” |
| daily commute | 过正式或过复杂都难坚持 | 稳定舒适的基础版型 | 每天都要精细维护 | 设定 5 分钟可出门 |
| date scenario + clean goal | 过度职场化显距离 | 柔和材质 + 干净轮廓 | 全套会议感 | 加一个柔和元素 |
| content creator + memorable goal | 记忆点过多显杂乱 | 一个主记忆点 + 稳定基底 | 多个强元素堆叠 | 保留品牌识别点 |
