# Outfit Taxonomy

Outfit taxonomy defines starter dimensions for clothing recommendations.

中文说明：
穿搭 taxonomy 用于描述版型、面料、领口、肩线、腰线、长度、正式度、场景和个人品牌目标。

- `silhouette_slightly_structured`
- `silhouette_relaxed`
- `silhouette_a_line`
- `silhouette_layered`
