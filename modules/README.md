Modules are the modular knowledge structure of StyleOS Protocol.

中文说明：
`modules/` 用来组织 StyleOS 的模块化知识结构。每个模块沉淀 taxonomy、tags、rule index、mapping matrix，以及未来 Pro library 的边界说明。

## Module Structure

Each module may include:

- taxonomy: module-level concepts and tag groups.
- tags: starter labels that compatible tools can read or extend.
- rule index: public starter rules for protocol demonstration.
- mapping matrix: signal-to-strategy examples for creator workflows.
- future Pro library boundary: what belongs to future commercial expert libraries or verified systems.

## Starter Content Notice

Current module content is starter content for protocol demonstration and community iteration.

中文说明：
当前模块内容不是专家认证库，不代表科学定论，也不代表数据验证结论。所有 starter rules 都应经过人工复核、社区反馈和后续 review 才能进入更高等级。

## Open-Core Boundary

The open repository includes starter rules, schemas, templates, synthetic cases, and contribution structures.

StyleOS Pro, advanced expert model libraries, real verified case libraries, commercial report generators, certification systems, and trademark assets are not part of this open-source repository.
