# Good First Issues

This document lists issue ideas for new contributors. It does not create GitHub Issues.

1. Add a synthetic hairstyle case for a low-maintenance user.
2. Add a synthetic hairstyle case for livestream preparation.
3. Improve the color taxonomy with starter tags.
4. Improve outfit scenario tags.
5. Translate glossary terms between English and Chinese.
6. Add a creator workflow example for a small styling studio.
7. Improve the evaluation checklist for lite reports.
8. Add a scenario rule card for interview preparation.
9. Add a synthetic case for client meeting styling.
10. Improve the barber brief field notes.
11. Add a makeup artist brief synthetic example.
12. Add an outfit styling brief synthetic example.
13. Review internal Markdown links in one module.
14. Add a FAQ entry for creator privacy.
15. Improve the open-core boundary explanation.
16. Add a rule-card example for maintenance constraints.
17. Add a synthetic case for glasses and hairstyle.
18. Improve the creator service menu template.
19. Add a developer example for preserving unknown tags.
20. Improve the Alpha feedback form questions.

## Contribution Rules

- Use synthetic examples.
- Keep changes focused.
- Do not add app code or dependencies.
- Do not include real user data.
- Link related schema or rule files.
