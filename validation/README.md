# Validation

Validation documents how StyleOS starter rules can move from protocol demonstration toward higher confidence levels.

中文说明：
`validation/` 用来定义证据等级、规则状态和 review 流程。当前 v0.1.2 Seed Knowledge Pack 大部分内容是 E0 starter content，不应被包装成专家认证。

## Files

- [Evidence Levels](evidence-levels.md)
- [Rule Status](rule-status.md)
- [Review Process](review-process.md)

## Core Principle

Validation should increase confidence without hiding uncertainty.

Rules should be marked clearly as starter, community-submitted, under-review, expert-reviewed, data-supported, deprecated, or pro-only.
