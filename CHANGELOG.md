# Changelog

All notable changes to StyleOS Protocol will be documented in this file.

## v0.1.0 - Protocol Skeleton

Initial open-source skeleton for StyleOS Protocol.

### Added

- Project README with positioning, architecture, scope, quick starts, data policy, roadmap, contribution guidance, license, and trademark notice.
- Core governance documents: `COMMERCIAL.md`, `TRADEMARKS.md`, `DATA_POLICY.md`, `CONTRIBUTOR_TERMS.md`, `CONTRIBUTING.md`, `PRIVACY.md`, and `SECURITY.md`.
- Protocol docs for input schema, style tag schema, rule card schema, lite report schema, and evaluation framework.
- Creator Kit templates for fan intake, service menu, lite report delivery, and creator workflow.
- Synthetic examples for one intake case, one rule card, and one lite report.
- GitHub issue forms and pull request template.

### Notes

- This version is documentation-only.
- This version does not include an app, SDK, API server, frontend framework, package manifest, or external dependency.
- All examples are synthetic and must not be treated as real user cases.
