# StyleOS vs Photo Studio

Photo studios and StyleOS solve different layers of the personal image workflow.

## Photo Studio

A photo studio usually solves a one-time photo result:

- makeup and styling for a shoot
- lighting
- pose guidance
- retouching
- final image delivery

## StyleOS

StyleOS solves reusable personal styling decision workflows:

- intake
- tags
- rule cards
- lite reports
- execution cards
- feedback
- creator service structure

## Relationship

Photo studios can become execution partners.

StyleOS is the upstream decision layer. It can help define what the user wants before the shoot and convert the goal into practical briefs.
