# Privacy

StyleOS Protocol is a public documentation repository. Do not use it to store or share private user data.

## Do Not Submit

- real user photos;
- real names;
- phone numbers;
- email addresses;
- social media handles;
- exact addresses;
- account IDs;
- private conversation logs;
- unauthorized service records;
- identifiable before-and-after cases;
- any data that can reasonably identify a person.

中文说明：
不要在本仓库提交真实用户照片、真实姓名、联系方式、账号、地址、私聊记录、未经授权的服务案例或任何可识别个人身份的信息。

## Accepted Examples

This repository accepts:

- synthetic examples;
- fully anonymized cases with authorization;
- abstracted rule cards that do not expose private source details;
- generic creator workflows;
- schema suggestions without personal data.

## If You Find Personal Data

If you find personal data in an issue, pull request, commit, or document:

1. Do not quote or spread the data.
2. Report it through the security contact process in [SECURITY.md](SECURITY.md).
3. Maintainers should remove or redact the content as soon as practical.

## Relationship to Commercial Products

Commercial StyleOS products may have separate privacy terms, consent flows, retention rules, and security processes. Those commercial terms are outside this open repository unless explicitly linked by ruhang365.
