# Creator Kit

The Creator Kit helps styling creators, outfit creators, image consultants, hairstylists, makeup artists, photography studios, and small beauty / styling businesses use StyleOS Protocol in practical services.

中文说明：
Creator Kit 是给创作者和小型服务团队使用的轻量工具包，用来收集粉丝资料、设计服务菜单、输出轻量报告、复盘规则。

## Included Templates

- [Fan Intake Form](fan-intake-form.md): collect structured fan or client inputs.
- [Service Menu Template](service-menu-template.md): design service packages.
- [Report Lite Template](report-lite-template.md): write a copy-ready lite report.
- [Creator Workflow](creator-workflow.md): run the complete workflow from intake to feedback.

## Use Boundaries

- Use synthetic examples for public sharing.
- Do not upload real user photos to this repository.
- Do not submit private user data.
- Review AI-generated output manually before delivery.
- Keep paid service promises clear and realistic.

## Suggested Workflow

1. Collect fan information with the intake form.
2. Tag the profile with StyleOS tags.
3. Select relevant rule cards.
4. Draft a lite report.
5. Review manually.
6. Deliver to the fan.
7. Collect feedback.
8. Abstract reusable rule improvements.
